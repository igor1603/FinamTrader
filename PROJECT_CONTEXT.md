## ФИНАМ API — КЛЮЧЕВЫЕ ДЕТАЛИ

**Авторизация — двухэтапная:**
1. `POST /v1/sessions` с телом `{"secret": "tapi_sk_..."}` → получаем JWT
2. JWT живёт **15 минут** — автообновление реализовано в `FinamAuthClient`
3. Все запросы: заголовок `Authorization: Bearer {jwt}`

**Важные ограничения:**
- Лимит запросов: 200 в минуту на метод
- Перерыв сервиса: 05:00–06:15 по Москве ежедневно
- Минимальный таймфрейм свечей: M1 (1 минута)
- Тиковая история недоступна через REST

**Реализованные эндпоинты:**
- `POST /v1/sessions` — авторизация ✅
- `GET /v1/accounts/{account_id}` — данные счёта ✅

**Структура ответа GET /v1/accounts/{account_id} для FORTS счёта:**
```json
{
  "account_id": "143047",
  "type": "FORTS",
  "status": "ACCOUNT_ACTIVE",
  "equity": {"value": "0.0"},
  "unrealized_profit": {"value": "0.0"},
  "positions": [],
  "cash": [],
  "portfolio_forts": {
    "available_cash": {"value": "0.0"},
    "money_reserved": {"value": "0.0"}
  }
}
```
*Числа приходят как объект {"value": "число_строкой"}*

**Тикеры:**
- Акции Мосбиржи: `SBER@MISX`, `VTBR@MISX`
- Фьючерсы FORTS: `BRM6@RTSX` (нефть Brent июнь 2026)
- Формат фьючерса: BASE + месяц (F=янв,G=фев,H=мар,J=апр,K=май,M=июн,N=июл,Q=авг,U=сен,V=окт,X=ноя,Z=дек) + год

## ФЬЮЧЕРС BRM6 — СПЕЦИФИКАЦИЯ

- **Полное название:** BR-6.26, расчётный фьючерс на нефть сорта Brent
- **Тикер на Финам:** `BRM6@RTSX`
- **Дата экспирации:** 01.06.2026
- **Размер контракта:** 10 баррелей
- **Котировка:** в USD за баррель
- **Биржа:** Московская биржа, секция FORTS (RTSX)
- **Текущая цена (май 2026):** ~110-111 USD
- **Есть мини-контракт:** `BMM6@RTSX` (1 баррель, гарантийное обеспечение ~1258 руб.)
